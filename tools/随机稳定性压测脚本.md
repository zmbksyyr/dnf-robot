# Stability Test Command

Use this script for all VM debug, smoke, pressure, and long stability tests.

Run on the VM:

```sh
python /root/vm_random_stability.py --hours 1
```

The script runs the full test set, injects destructive cases, backs up and restores needed data, and writes a report.

Result files:

```sh
ls -td /root/robot_stability_* | head -n 1
cat /root/robot_stability_*/report.md
cat /root/robot_stability_*/failures.json
```
